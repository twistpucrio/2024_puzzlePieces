Disney Puzzle

Projeto 2 - TWIST

Estamos desenvolvendo um jogo de Quebra-Cabeça "Slide Puzzle" como um challenge de programação envolvendo HTML, CSS e JS. Dessa forma, os requisitos do projeto são Tags semânticas, CSS externo, JS externo, e ter 3 níveis de jogabilidade: fácil, médio e difícil. Nós escolhemos o tema Disney para nos basear, onde cada nível terá um personagem de acordo com o que decidimos, no primeiro nível teria uma princesa, no segundo teria um personagem mais neutro, e no terceiro teria um vilão.
